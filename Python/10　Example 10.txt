10-1 數軸分段問題:

(1) Input :使用者輸入日期資訊(西元年、月份及日期)
    Output:顯示日期為該年度之第幾天

--------------------------------------------------------------------------------

year =int(input("輸入西元年:"))
month=int(input("請輸入月份:"))
date =int(input("請輸入日期:"))
day=0

norm=[31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31]

#閏年規則:逢四則閏、逢百不閏、逢四百再閏
if year%4!=0:
    print("%d年為平年"%(year))

#四的倍數為平年:必為100之倍數而不整除400
elif year%100==0 and year%400!=0:
    print("%d年為平年"%(year))
else:
    norm[1]=29
    print("%d年為閏年"%(year))

#月份為N，僅需累加至N-1月，即第N-2項
for i in range(0, month-1):
    day+=norm[i]
day+=date
print("%d月%d日是%d年的第%d天"%(month, date, year, day))

(2) Input :公司對於績效分數之規定如下(1)15件(含)以下者，每案累積1點積分        ，使用者自由輸入完成案件數
                                     (2)25件(含)以下者，高出部分每案累積2點積分
                                     (3)30件(含)以下者，高出部分每案累積3點積分
                                     (4)高出30件之部分，每案累積5點積分
    Output:顯示該完案數可獲得之績效分數

--------------------------------------------------------------------------------
 
case=int(input("請輸入完成案件數:"))
#數軸分段，case將與下一項比較，列表需多寫末項(必小於而跳出迴圈)
caselist=[0, 15, 25, 30, case+1]
pointlist=[1, 2, 3, 5]
point=0

for i in range(len(caselist)):
    #若超過下一項，則累得該段所有點數＝區間長度*分值
    if case>=caselist[i+1]:
        point+=(caselist[i+1]-caselist[i])*pointlist[i]
    #若未達下一項，則結算本段所累點數＝截軸長度*分值，結束迴圈
    else:
        point+=(case-caselist[i])*pointlist[i]
        break

print("完案數%d，可獲得績效%d點"%(case, point))

--------------------------------------------------------------------------------

day=int(input("請輸入獼猴吃桃日數:"))

(2) Input :五隻猴子分桃，先丟棄一顆後取走五等分之一份；再交由下位重複執行此步
　　Output:最後所有猴子均可取走整數顆桃子，求桃數至少為何

--------------------------------------------------------------------------------

#取完後剩五分之四，故倒算可知:取前之桃數為取後1.25倍+1
peach=1
monkey=1
trys=1

#第N隻猴子執行分桃工作，當五隻均成功時即輸出
while monkey<6:
    #trys可迴圈執行，桃數遞迴倒推
    if int(1.25*trys)==1.25*trys:
        trys=1.25*trys+1
        monkey+=1
    else:
　　　　#每當分桃失敗，peach往下累計而猴子清零
        peach+=1
        monkey=1
　　　　#trys與peach分開以防遞迴桃數跳過某些值
        trys=peach

print("桃數至少為%d"%(trys))
